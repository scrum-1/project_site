Pyslvs 編譯進度

界面調整 - 面板

當初構想的清單原本也是置頂的視窗，但是後來發現在 Windows 中開啟會導致清單視窗 crash，所以將一些額外的面板合併到點座標的分頁去。

不過這些面板插入分頁後排版不太人性化，之後還會再修正。

編譯 - Makefile

最後還是統一兩個版本都使用 Makefile 來編譯，方便 Windows 可攜的版本下指令，只要使用下列語句即可分辨平台：

ifeq ($(OS),Windows_NT)
#執行 Windows 編譯
else
#執行 Linux 編譯
endif

另外在 Readme 上也有做修正。
